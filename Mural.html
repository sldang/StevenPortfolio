    Project Five

    A Creative Design Project

    Skills

    Swift, 3D Modeling, UI/UX Design

    Tools

    Code, Figma, Blender, Reality Composer Pro

    Overview

    Utilizing the real-life mural illustrated at USC’s Iovine and Young Hall, we created a Scavenger Hunt and Message Board to accompany the mural. We first started with figuring out User Flows and how a user might want to replay the scavenger hunt. Then we 3D modeled items, in Blender, that would be included in the scavenger hunt, where the player would throw the Airplane to collect the items. We then used XCode to code the experience for both IOS and VisionOS with the help of Reality Composer Pro.
